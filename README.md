### 🔍 What's Inside?
This repo serves as the architectural demo of the foundation I use to build production bots. It is designed to help you skip repetitive setup and start writing business logic immediately.

*   **Modular Design:** Separated core logic from business modules.
*   **Plugin System:** Extend your bot without touching core code.
*   **Safe Logging:** Pre-configured Python logging module.

*   **Core:** The engine that handles the system lifecycle.

*   **Plugins:** Dynamic manager to add features on the fly.
